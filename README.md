# GMI - 精神疾病指南

这是一个基于 MkDocs 构建的精神疾病指南网站，提供关于抑郁症、焦虑症、躁狂症、双向情感障碍、人格分裂和精神分裂症的详细信息。

## 功能特点

- 📚 详细的疾病介绍和症状分级
- 💊 全面的治疗方法和资源
