## 重要声明

⚠️ **免责声明**：本指南仅供教育和信息目的，不能替代专业医疗建议、诊断或治疗。如果您或您认识的人正在经历精神健康问题，请立即寻求专业医疗帮助。

!!! danger "紧急情况 - 立即寻求帮助"
    如果您或他人出现以下情况，请立即采取行动：
    
    **自杀风险征象：**
    - 明确表达自杀想法或计划
    - 谈论死亡、自杀或伤害自己
    - 寻找自杀方法或工具
    - 告别行为或整理遗物
    - 极度绝望或无助感
    - 突然的情绪平静（可能表示已做决定）
    
    **伤害他人风险：**
    - 威胁伤害他人
    - 暴力行为或冲动
    - 妄想他人要伤害自己
    - 严重的激越或攻击性
    
    **严重精神症状：**
    - 严重的幻觉或妄想
    - 完全脱离现实
    - 严重的意识混乱
    - 无法照顾基本生活需求
    
    **立即行动：**
    - 🚨 拨打紧急电话：**120**（中国大陆急救）
    - 📞 全国心理危机干预热线：**400-161-9995**
    - 👥 青少年心理咨询热线：**12355**
    - 🏥 白天：立即前往医院精神科或急诊科
    - 🌙 夜晚：前往急诊科或寻求朋友/家人帮助
    - 👮 必要时联系警察：**110**

!!! warning "药物安全警告"
    **服用精神科药物期间，如出现以下情况请立即就医：**
    
    - 严重过敏反应（皮疹、呼吸困难、面部肿胀）
    - 5-羟色胺综合征（高热、肌肉僵硬、意识改变）
    - 恶性综合征（高热、肌肉僵硬、自主神经紊乱）
    - 严重心律失常（胸痛、心悸、晕厥）
    - 锂中毒症状（严重恶心呕吐、意识模糊、肌肉抽搐）
    - 严重的锥体外系反应（肌肉痉挛、呼吸困难）
    
    **重要提醒：**
    - 不要自行停药或调整剂量
    - 避免饮酒和使用非法药物
    - 告知医生所有正在使用的药物
    - 妊娠期和哺乳期需要特别注意

!!! info "何时寻求专业帮助"
    **以下情况建议咨询专业医生：**
    
    **症状持续性：**
    - 抑郁、焦虑症状持续2周以上
    - 情绪问题影响日常生活和工作
    - 睡眠、食欲持续异常
    
    **功能损害：**
    - 无法正常工作或学习
    - 人际关系严重受影响
    - 日常自理能力下降
    
    **家人朋友担忧：**
    - 周围人表达对您的担心
    - 行为或性格明显改变
    - 社交回避或孤立
    
    **既往病史：**
    - 有精神疾病家族史
    - 曾经有过精神健康问题
    - 正在服用精神科药物
